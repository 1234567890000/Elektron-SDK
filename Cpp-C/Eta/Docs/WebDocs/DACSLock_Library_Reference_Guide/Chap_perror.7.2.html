Synopsis
The DACS_perror() function creates a textual message describing the last error generated by a DACS Library call. This message is saved to the supplied buffer. The error number is taken from the location Dacs_Error -> dacs_errno variable pointed to by the user application, when a library function returns a DACS_FAILURE.
 
 
DACS_perror (err_buffer, buffer_len, text, Dacs_Error);
    unsigned char        *err_buffer;
    int                  buffer_len;
    unsigned char        *text;
    DACS_ERROR_TYPE      *Dacs_Error;
