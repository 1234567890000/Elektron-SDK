7.6
Specific Usage: Enterprise Platform for Real-Time
For the most part, MarketByOrder data from the Enterprise Platform for Real-Time is the same as it is from the original source of the data (e.g. RDF Direct). However, if caching is enabled in an Enterprise Platform for Real-Time component, there are two differences.
•
The number of messages packed into each Refresh response message may be different.
•
An updated reponse message might be delivered between Refresh response messages and before the message with RefreshMsg.Complete set true. It is the consumer applications responsibility to apply the indicated changes.
