12.2
ItemList
The client application can specify interest in multiple items by using a single batch request message. To do this, encode a list of item names in the request message payload. This is supported on any non-administrative RDM and any user defined DMM. For further details, refer to EMA 3.0 Developers Guide C++ Edition.
For batch request messages, the payload contains at a minimum an element list which includes the following element entry:
